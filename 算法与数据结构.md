## 一、数组、链表、栈、队列

### 1. 数组 vs 链表核心对比

```
数组（Array）— 连续内存
┌──┬──┬──┬──┬──┬──┐
│0 │1 │2 │3 │4 │5 │  固定大小，随机访问 O(1)
└──┴──┴──┴──┴──┴──┘

链表（LinkedList）— 离散内存
[val|next] → [val|next] → [val|next] → null
  随机访问 O(n)，头插/尾插 O(1)
```

| 操作 | 数组 | 链表 |
|---|---|---|
| 随机访问 `get(i)` | O(1) | O(n) |
| 头部插入/删除 | O(n)（整体移位）| O(1) |
| 尾部插入 | O(1)（均摊）| O(1)（有tail指针）|
| 中间插入/删除 | O(n) | O(n)（查找）+ O(1)（操作）|
| 空间 | 连续，Cache 友好 | 额外 next 指针，Cache 不友好 |

### 2. 链表高频操作手写

#### 反转链表（字节必考）

```java
// 迭代法（推荐，O(n) 时间，O(1) 空间）
public ListNode reverseList(ListNode head) {
    ListNode prev = null, cur = head;
    while (cur != null) {
        ListNode next = cur.next;   // 保存后继节点
        cur.next = prev;            // 反转指针
        prev = cur;                 // prev 前进
        cur = next;                 // cur 前进
    }
    return prev;  // prev 指向新头节点
}

// 递归法（理解递归栈）
public ListNode reverseList(ListNode head) {
    if (head == null || head.next == null) return head;
    ListNode newHead = reverseList(head.next);  // 假设后面已反转
    head.next.next = head;   // 后继节点指回自己
    head.next = null;        // 断开原来的指向
    return newHead;
}
```

#### 检测环并找入口（Floyd 快慢指针）

```java
public ListNode detectCycle(ListNode head) {
    ListNode slow = head, fast = head;
    // Phase 1: 判断是否有环
    while (fast != null && fast.next != null) {
        slow = slow.next;
        fast = fast.next.next;
        if (slow == fast) break;   // 相遇 → 有环
    }
    if (fast == null || fast.next == null) return null;  // 无环
    
    // Phase 2: 从 head 和相遇点各走一步，再次相遇即入口
    // 原理：head 到入口距离 = 相遇点到入口距离
    slow = head;
    while (slow != fast) {
        slow = slow.next;
        fast = fast.next;
    }
    return slow;
}
```

#### 合并两个有序链表

```java
public ListNode mergeTwoLists(ListNode l1, ListNode l2) {
    ListNode dummy = new ListNode(0), cur = dummy;
    while (l1 != null && l2 != null) {
        if (l1.val <= l2.val) { cur.next = l1; l1 = l1.next; }
        else                   { cur.next = l2; l2 = l2.next; }
        cur = cur.next;
    }
    cur.next = (l1 != null) ? l1 : l2;
    return dummy.next;
}
```

#### 找链表中点（快慢指针）

```java
// 偶数长度返回后中点（奇数长度返回中点）
public ListNode middleNode(ListNode head) {
    ListNode slow = head, fast = head;
    while (fast != null && fast.next != null) {
        slow = slow.next;
        fast = fast.next.next;
    }
    return slow;
}
```

### 3. 单调栈

**核心思想：** 维护一个单调递增/递减的栈，用 O(n) 解决"找下一个更大/更小元素"类问题。

```java
// 模板：找每个元素右边第一个比它大的元素（下一个更大元素）
public int[] nextGreaterElement(int[] nums) {
    int n = nums.length;
    int[] res = new int[n];
    Arrays.fill(res, -1);
    Deque<Integer> stack = new ArrayDeque<>();  // 存下标，单调递减栈

    for (int i = 0; i < n; i++) {
        // 当前元素比栈顶大 → 栈顶的"下一个更大元素"就是当前元素
        while (!stack.isEmpty() && nums[i] > nums[stack.peek()]) {
            res[stack.pop()] = nums[i];
        }
        stack.push(i);
    }
    return res;
}
```

**字节高频题：** 接雨水、柱状图最大矩形、每日温度

```java
// 接雨水（单调递减栈）
public int trap(int[] height) {
    Deque<Integer> stack = new ArrayDeque<>();
    int water = 0;
    for (int i = 0; i < height.length; i++) {
        while (!stack.isEmpty() && height[i] > height[stack.peek()]) {
            int mid = stack.pop();
            if (!stack.isEmpty()) {
                int h = Math.min(height[stack.peek()], height[i]) - height[mid];
                int w = i - stack.peek() - 1;
                water += h * w;
            }
        }
        stack.push(i);
    }
    return water;
}
```

### 4. 单调队列（滑动窗口最大值）

```java
// 字节高频：滑动窗口最大值 O(n)
public int[] maxSlidingWindow(int[] nums, int k) {
    Deque<Integer> deque = new ArrayDeque<>();  // 存下标，队头是窗口最大值下标
    int[] res = new int[nums.length - k + 1];
    
    for (int i = 0; i < nums.length; i++) {
        // 移除超出窗口的队头
        while (!deque.isEmpty() && deque.peekFirst() < i - k + 1) {
            deque.pollFirst();
        }
        // 移除所有比当前元素小的队尾（维护单调递减）
        while (!deque.isEmpty() && nums[deque.peekLast()] < nums[i]) {
            deque.pollLast();
        }
        deque.offerLast(i);
        if (i >= k - 1) res[i - k + 1] = nums[deque.peekFirst()];
    }
    return res;
}
```

### 5. 面试标准答法

> 数组随机访问 O(1)，但插入删除需要移动元素，适合"读多写少"场景；链表插入删除 O(1)，但随机访问 O(n)，适合"频繁增删"场景。链表还有缓存局部性差的问题（节点离散，CPU Cache 命中率低）。单调栈/单调队列是解决"区间最值"和"下一个更大/小元素"问题的常用技巧，核心是**用栈/队列维护单调性，空间换时间**，将暴力 O(n²) 降到 O(n)。

### 6. 常见追问

**Q: Java 中 LinkedList vs ArrayList，各自适用场景？**
> ArrayList 底层数组，随机访问 O(1)，尾部追加均摊 O(1)，但扩容需要复制（O(n)），中间插删 O(n)。LinkedList 底层双向链表，两端操作 O(1)（可当 Deque 用），随机访问 O(n)，单个节点有 prev/next 两个指针的额外空间开销。实际工程中 ArrayList 用途更广，因为大多数场景读多写少，且 CPU Cache 命中率更高。LinkedList 只在频繁头尾操作（队列/双端队列）时才有优势，但通常 ArrayDeque 比 LinkedList 性能更好（局部性）。

## 二、二叉树

### 1. 树的基本概念

```
          1          ← 根节点（root）
        /   \
       2     3       ← 高度 2
      / \   /
     4   5 6         ← 叶节点

深度（depth）：从根到该节点的边数
高度（height）：从该节点到最深叶子的边数
满二叉树：每个节点有 0 或 2 个子节点
完全二叉树：除最后一层外都满，最后一层从左向右填充（堆用此结构）
```

### 2. 遍历（递归 + 迭代两种写法）

#### 前序遍历（根 → 左 → 右）

```java
// 递归
void preorder(TreeNode root, List<Integer> res) {
    if (root == null) return;
    res.add(root.val);
    preorder(root.left, res);
    preorder(root.right, res);
}

// 迭代（栈）— 字节偏好手写迭代
public List<Integer> preorderTraversal(TreeNode root) {
    List<Integer> res = new ArrayList<>();
    Deque<TreeNode> stack = new ArrayDeque<>();
    if (root != null) stack.push(root);
    while (!stack.isEmpty()) {
        TreeNode node = stack.pop();
        res.add(node.val);
        if (node.right != null) stack.push(node.right);  // 先压右
        if (node.left != null)  stack.push(node.left);   // 再压左（先出）
    }
    return res;
}
```

#### 中序遍历（左 → 根 → 右）— BST 输出有序

```java
// 迭代（模拟递归调用栈）
public List<Integer> inorderTraversal(TreeNode root) {
    List<Integer> res = new ArrayList<>();
    Deque<TreeNode> stack = new ArrayDeque<>();
    TreeNode cur = root;
    while (cur != null || !stack.isEmpty()) {
        while (cur != null) {           // 一路向左压栈
            stack.push(cur);
            cur = cur.left;
        }
        cur = stack.pop();              // 弹出，访问
        res.add(cur.val);
        cur = cur.right;                // 转向右子树
    }
    return res;
}
```

#### 后序遍历（左 → 右 → 根）

```java
// 迭代（前序变形：根→右→左，最后反转）
public List<Integer> postorderTraversal(TreeNode root) {
    LinkedList<Integer> res = new LinkedList<>();
    Deque<TreeNode> stack = new ArrayDeque<>();
    if (root != null) stack.push(root);
    while (!stack.isEmpty()) {
        TreeNode node = stack.pop();
        res.addFirst(node.val);          // 头插 = 最后反转
        if (node.left != null)  stack.push(node.left);
        if (node.right != null) stack.push(node.right);
    }
    return res;
}
```

#### 层序遍历（BFS）

```java
public List<List<Integer>> levelOrder(TreeNode root) {
    List<List<Integer>> res = new ArrayList<>();
    if (root == null) return res;
    Queue<TreeNode> queue = new ArrayDeque<>();
    queue.offer(root);
    while (!queue.isEmpty()) {
        int size = queue.size();         // 当前层节点数
        List<Integer> level = new ArrayList<>();
        for (int i = 0; i < size; i++) {
            TreeNode node = queue.poll();
            level.add(node.val);
            if (node.left != null)  queue.offer(node.left);
            if (node.right != null) queue.offer(node.right);
        }
        res.add(level);
    }
    return res;
}
```

### 3. BST（二叉搜索树）

**核心性质：** 左子树所有节点 < 根 < 右子树所有节点，**中序遍历结果有序**。

```
     5
    / \
   3   7       中序：1 2 3 4 5 6 7 8 9（有序！）
  / \ / \
 2  4 6  9
/       /
1      8
```

```java
// BST 查找：O(log n) 平均，O(n) 最坏（退化成链表）
public TreeNode search(TreeNode root, int val) {
    if (root == null || root.val == val) return root;
    return val < root.val ? search(root.left, val) : search(root.right, val);
}

// BST 插入
public TreeNode insert(TreeNode root, int val) {
    if (root == null) return new TreeNode(val);
    if (val < root.val)       root.left  = insert(root.left,  val);
    else if (val > root.val)  root.right = insert(root.right, val);
    return root;
}

// BST 验证（中序遍历检查是否递增）
public boolean isValidBST(TreeNode root) {
    long prev = Long.MIN_VALUE;
    Deque<TreeNode> stack = new ArrayDeque<>();
    TreeNode cur = root;
    while (cur != null || !stack.isEmpty()) {
        while (cur != null) { stack.push(cur); cur = cur.left; }
        cur = stack.pop();
        if (cur.val <= prev) return false;   // 不满足严格递增
        prev = cur.val;
        cur = cur.right;
    }
    return true;
}
```

### 4. 红黑树原理（面试原理题）

```
红黑树 5 条性质：
1. 每个节点是红色或黑色
2. 根节点是黑色
3. 所有 NULL 叶节点（哨兵）是黑色
4. 红色节点的子节点必须是黑色（不能连续两个红节点）
5. 从任意节点到其叶节点的所有路径，黑色节点数量相同（黑色高度一致）

插入调整（新节点默认红色，以免破坏性质5）：
- Case1：叔父节点为红色 → 变色（父+叔变黑，祖父变红，递归向上）
- Case2：叔父节点为黑色，LL或RR型 → 一次旋转+变色
- Case3：叔父节点为黑色，LR或RL型 → 先旋转成LL/RR，再Case2

树高保证：O(log n)（黑高不超过 log n，总高不超过 2 × 黑高）
```

**Java 中的红黑树：** `TreeMap`、`TreeSet` 底层实现；`HashMap` 链表长度 ≥8 + 数组长度 ≥64 时转为红黑树。

### 5. 字节高频二叉树题

```java
// 二叉树最大深度
public int maxDepth(TreeNode root) {
    if (root == null) return 0;
    return 1 + Math.max(maxDepth(root.left), maxDepth(root.right));
}

// 对称二叉树
public boolean isSymmetric(TreeNode root) {
    return check(root.left, root.right);
}
boolean check(TreeNode l, TreeNode r) {
    if (l == null && r == null) return true;
    if (l == null || r == null || l.val != r.val) return false;
    return check(l.left, r.right) && check(l.right, r.left);
}

// 路径总和（root到叶子）
public boolean hasPathSum(TreeNode root, int target) {
    if (root == null) return false;
    if (root.left == null && root.right == null) return root.val == target;
    return hasPathSum(root.left,  target - root.val)
        || hasPathSum(root.right, target - root.val);
}

// 二叉树最低公共祖先（LCA）
public TreeNode lowestCommonAncestor(TreeNode root, TreeNode p, TreeNode q) {
    if (root == null || root == p || root == q) return root;
    TreeNode left  = lowestCommonAncestor(root.left,  p, q);
    TreeNode right = lowestCommonAncestor(root.right, p, q);
    if (left != null && right != null) return root;  // p 和 q 分属两侧
    return left != null ? left : right;
}
```

### 6. 面试标准答法

> 二叉树遍历分为深度优先（前/中/后序）和广度优先（层序）。实际面试中**迭代写法**更考察代码功底：前序用栈（先压右再压左），中序用栈模拟调用帧（一路向左），后序是前序调整后反转。BST 的核心性质是中序有序，查找/插入 O(log n) 平均，但可能退化到 O(n)，红黑树通过旋转+变色保证树高始终 O(log n)，是自平衡 BST 的工程实现首选（Java TreeMap/HashMap 链表转换都用红黑树）。

## 三、动态规划

### 1. DP 三要素

```
① 状态定义：dp[i] 或 dp[i][j] 代表什么？
② 转移方程：dp[i] 由哪些"子问题"推导出来？
③ 初始化&边界：base case 是什么？
```

### 2. 一维 DP 经典题

#### 爬楼梯（字节必考）

```java
// dp[i] = 到达第 i 阶的方法数
// dp[i] = dp[i-1] + dp[i-2]
public int climbStairs(int n) {
    if (n <= 2) return n;
    int a = 1, b = 2;
    for (int i = 3; i <= n; i++) {
        int c = a + b;
        a = b; b = c;
    }
    return b;  // 空间 O(1) 优化
}
```

#### 打家劫舍

```java
// dp[i] = 偷前 i 间房子的最大金额
// dp[i] = max(dp[i-1], dp[i-2] + nums[i])  （不偷 vs 偷当前）
public int rob(int[] nums) {
    int n = nums.length;
    if (n == 1) return nums[0];
    int prev2 = nums[0], prev1 = Math.max(nums[0], nums[1]);
    for (int i = 2; i < n; i++) {
        int cur = Math.max(prev1, prev2 + nums[i]);
        prev2 = prev1; prev1 = cur;
    }
    return prev1;
}
```

#### 最长递增子序列（LIS）

```java
// dp[i] = 以 nums[i] 结尾的最长递增子序列长度
// O(n²) DP，O(n log n) 贪心+二分优化
public int lengthOfLIS(int[] nums) {
    int n = nums.length, maxLen = 1;
    int[] dp = new int[n];
    Arrays.fill(dp, 1);
    for (int i = 1; i < n; i++) {
        for (int j = 0; j < i; j++) {
            if (nums[j] < nums[i]) dp[i] = Math.max(dp[i], dp[j] + 1);
        }
        maxLen = Math.max(maxLen, dp[i]);
    }
    return maxLen;
}

// O(n log n) 优化：维护单调递增的 tails 数组（贪心+二分）
public int lengthOfLIS_Optimized(int[] nums) {
    List<Integer> tails = new ArrayList<>();
    for (int num : nums) {
        // 二分找 tails 中第一个 >= num 的位置
        int lo = 0, hi = tails.size();
        while (lo < hi) {
            int mid = (lo + hi) / 2;
            if (tails.get(mid) < num) lo = mid + 1; else hi = mid;
        }
        if (lo == tails.size()) tails.add(num);
        else tails.set(lo, num);
    }
    return tails.size();
}
```

### 3. 二维 DP 经典题

#### 编辑距离（字节高频）

```java
// dp[i][j] = word1 前 i 个字符转换成 word2 前 j 个字符所需最少操作数
// 操作：插入、删除、替换
public int minDistance(String word1, String word2) {
    int m = word1.length(), n = word2.length();
    int[][] dp = new int[m + 1][n + 1];
    // 初始化：空串转换
    for (int i = 0; i <= m; i++) dp[i][0] = i;
    for (int j = 0; j <= n; j++) dp[0][j] = j;
    
    for (int i = 1; i <= m; i++) {
        for (int j = 1; j <= n; j++) {
            if (word1.charAt(i-1) == word2.charAt(j-1)) {
                dp[i][j] = dp[i-1][j-1];              // 相同，不操作
            } else {
                dp[i][j] = 1 + Math.min(dp[i-1][j-1],  // 替换
                               Math.min(dp[i-1][j],      // 删除word1[i]
                                        dp[i][j-1]));    // 插入word2[j]
            }
        }
    }
    return dp[m][n];
}
```

#### 最长公共子序列（LCS）

```java
// dp[i][j] = text1 前 i 个字符和 text2 前 j 个字符的 LCS 长度
public int longestCommonSubsequence(String text1, String text2) {
    int m = text1.length(), n = text2.length();
    int[][] dp = new int[m + 1][n + 1];
    for (int i = 1; i <= m; i++) {
        for (int j = 1; j <= n; j++) {
            if (text1.charAt(i-1) == text2.charAt(j-1))
                dp[i][j] = dp[i-1][j-1] + 1;
            else
                dp[i][j] = Math.max(dp[i-1][j], dp[i][j-1]);
        }
    }
    return dp[m][n];
}
```

#### 0-1 背包

```java
// dp[j] = 容量为 j 时能装的最大价值（空间压缩到一维，逆序遍历）
public int knapsack(int[] weights, int[] values, int capacity) {
    int[] dp = new int[capacity + 1];
    for (int i = 0; i < weights.length; i++) {
        // 逆序：防止同一件物品被多次使用
        for (int j = capacity; j >= weights[i]; j--) {
            dp[j] = Math.max(dp[j], dp[j - weights[i]] + values[i]);
        }
    }
    return dp[capacity];
}
```

### 4. 面试标准答法

> DP 的本质是**带记忆的穷举**，避免重复计算子问题。三步走：① 明确状态（`dp[i]` 含义）；② 写出转移方程（当前状态如何由历史状态推出）；③ 处理初始值和边界（base case）。字节面试常见题：爬楼梯（斐波那契变形）、打家劫舍（线性DP）、编辑距离（二维DP）、最长递增子序列（可 O(n log n) 贪心优化）。空间优化技巧：如果 dp[i] 只依赖 dp[i-1]，可滚动数组或两个变量优化到 O(1)。

### 5. 常见追问

**Q: 贪心和动态规划的区别？**
> 贪心每步选当前最优，不回溯，适用于"局部最优即全局最优"（最小生成树、区间贪心、霍夫曼编码）。DP 穷举所有子问题，选最优，有重叠子问题和最优子结构两个特征。贪心更快（通常 O(n log n)），但适用范围窄。LIS 问题既有 O(n²) DP，也有 O(n log n) 贪心+二分组合解法。

## 四、回溯算法

### 1. 回溯模板

```java
// 通用框架
void backtrack(路径, 选择列表) {
    if (满足终止条件) {
        结果集.add(路径);
        return;
    }
    for (选择 : 选择列表) {
        做选择;           // 将选择加入路径
        backtrack(路径, 新的选择列表);
        撤销选择;         // 恢复现场
    }
}
```

```java
void backtrack(int[] nums, boolean[] used, List<Integer> path, List<List<Integer>> res) {
    if (path.size() == nums.length) {
        res.add(new ArrayList<>(path));   // 注意要复制
        return;
    }
    for (int i = 0; i < nums.length; i++) {
        if (used[i]) continue;
        used[i] = true;
        path.add(nums[i]);
        backtrack(nums, used, path, res);
        path.remove(path.size() - 1);    // 撤销
        used[i] = false;
    }
}
```

### 4. N 皇后

```java
public List<List<String>> solveNQueens(int n) {
    List<List<String>> res = new ArrayList<>();
    int[] queens = new int[n];    // queens[row] = col，表示第 row 行皇后放在第 col 列
    Arrays.fill(queens, -1);
    Set<Integer> cols = new HashSet<>(), diag1 = new HashSet<>(), diag2 = new HashSet<>();
    backtrack(n, 0, queens, cols, diag1, diag2, res);
    return res;
}

void backtrack(int n, int row, int[] queens,
               Set<Integer> cols, Set<Integer> diag1, Set<Integer> diag2,
               List<List<String>> res) {
    if (row == n) {
        res.add(generateBoard(queens, n));
        return;
    }
    for (int col = 0; col < n; col++) {
        // 检查列、左斜线（row-col）、右斜线（row+col）是否冲突
        if (cols.contains(col) || diag1.contains(row - col) || diag2.contains(row + col))
            continue;
        queens[row] = col;
        cols.add(col);  diag1.add(row - col);  diag2.add(row + col);
        backtrack(n, row + 1, queens, cols, diag1, diag2, res);
        queens[row] = -1;
        cols.remove(col);  diag1.remove(row - col);  diag2.remove(row + col);
    }
}

List<String> generateBoard(int[] queens, int n) {
    List<String> board = new ArrayList<>();
    for (int row = 0; row < n; row++) {
        char[] line = new char[n];
        Arrays.fill(line, '.');
        line[queens[row]] = 'Q';
        board.add(new String(line));
    }
    return board;
}
```

### 5. 面试标准答法

> 回溯是暴力穷举的优化版，通过**剪枝**（发现当前路径不可能产生有效结果时提前返回）减少搜索空间。三步模板：做选择 → 递归 → 撤销选择。时间复杂度通常是 O(n! ) 或 O(2^n)，优化关键在剪枝：如对数组排序后跳过重复元素、判断当前路径已超目标就不再递归。字节高频：全排列、子集、组合总和、N 皇后、单词搜索。

### 6. 常见追问

**Q: 回溯、DFS、递归三者的关系？**
> 三者都用递归实现，但概念层次不同。递归是编程技巧（函数调用自身）；DFS 是图/树的搜索策略（深度优先遍历，尽量向深处走）；回溯是在 DFS 基础上增加了"撤销选择"的能力，用于穷举所有可能性（如排列组合），属于有状态的 DFS。DFS 不需要撤销（因为树节点只访问一次），回溯必须撤销（因为路径上的选择会影响后续递归，需要恢复现场）。

### 2. 手写快速排序（字节必考）

```java
public void quickSort(int[] nums, int left, int right) {
    if (left >= right) return;
    int pivotIdx = partition(nums, left, right);
    quickSort(nums, left, pivotIdx - 1);
    quickSort(nums, pivotIdx + 1, right);
}

// 单边扫描 partition（lomuto 分区法）
private int partition(int[] nums, int left, int right) {
    // 随机选 pivot，防止已有序数组退化为 O(n²)
    int randIdx = left + (int)(Math.random() * (right - left + 1));
    swap(nums, randIdx, right);
    int pivot = nums[right];
    
    int i = left - 1;   // i 指向"小于 pivot 区域"的末尾
    for (int j = left; j < right; j++) {
        if (nums[j] <= pivot) {
            i++;
            swap(nums, i, j);
        }
    }
    swap(nums, i + 1, right);  // 将 pivot 放到正确位置
    return i + 1;
}

// 双指针 partition（hoare 分区法，更高效）
private int partitionHoare(int[] nums, int left, int right) {
    int pivot = nums[left + (right - left) / 2];
    int i = left - 1, j = right + 1;
    while (true) {
        do { i++; } while (nums[i] < pivot);
        do { j--; } while (nums[j] > pivot);
        if (i >= j) return j;
        swap(nums, i, j);
    }
}

private void swap(int[] nums, int a, int b) {
    int tmp = nums[a]; nums[a] = nums[b]; nums[b] = tmp;
}
```

**三路快排（处理大量重复元素，字节进阶问）：**

```java
// 荷兰国旗问题：将数组分为 < pivot | == pivot | > pivot 三段
void quickSort3Way(int[] nums, int lo, int hi) {
    if (lo >= hi) return;
    int pivot = nums[lo];
    int lt = lo, gt = hi, i = lo + 1;
    while (i <= gt) {
        if      (nums[i] < pivot) swap(nums, i++, lt++);
        else if (nums[i] > pivot) swap(nums, i, gt--);
        else                      i++;
    }
    // nums[lo..lt-1] < pivot, nums[lt..gt] == pivot, nums[gt+1..hi] > pivot
    quickSort3Way(nums, lo, lt - 1);
    quickSort3Way(nums, gt + 1, hi);
}
```

### 3. 手写归并排序

```java
public void mergeSort(int[] nums, int left, int right) {
    if (left >= right) return;
    int mid = left + (right - left) / 2;
    mergeSort(nums, left, mid);
    mergeSort(nums, mid + 1, right);
    merge(nums, left, mid, right);
}

private void merge(int[] nums, int left, int mid, int right) {
    // 临时数组存放已排序的结果
    int[] tmp = new int[right - left + 1];
    int i = left, j = mid + 1, k = 0;
    while (i <= mid && j <= right) {
        if (nums[i] <= nums[j]) tmp[k++] = nums[i++];
        else                    tmp[k++] = nums[j++];
    }
    while (i <= mid)   tmp[k++] = nums[i++];
    while (j <= right) tmp[k++] = nums[j++];
    // 写回原数组
    System.arraycopy(tmp, 0, nums, left, tmp.length);
}
```

**归并排序的应用：** 求逆序对数（归并时右边元素先归入，左边剩余 mid-i+1 个全都与它构成逆序对）。

### 4. 面试标准答法

> 面试重点是**快排和归并**。快排平均 O(n log n)，空间 O(log n)（递归栈），最坏 O(n²)（数组已有序时 pivot 每次选端点），优化手段是**随机选 pivot** 或**三路快排**（处理重复元素）。归并排序时间复杂度稳定 O(n log n)，空间 O(n)，是**稳定排序**（相同元素相对顺序不变），适合外排序（大文件排序，内存放不下时分块归并）和求逆序对。Java 对对象数组用 TimSort（插入排序+归并排序混合），对基本类型用双轴快排。

### 1. 滑动窗口模板

**适用场景：** 在连续子数组/子串中找满足条件的最长/最短/有多少个。

```java
// 不固定大小的滑动窗口（最长不重复子串为例）
public int lengthOfLongestSubstring(String s) {
    Map<Character, Integer> window = new HashMap<>();
    int left = 0, right = 0, maxLen = 0;
    
    while (right < s.length()) {
        char c = s.charAt(right++);
        window.merge(c, 1, Integer::sum);           // 扩大窗口
        
        while (window.get(c) > 1) {                 // 窗口条件不满足时收缩
            char d = s.charAt(left++);
            window.merge(d, -1, Integer::sum);
            if (window.get(d) == 0) window.remove(d);
        }
        maxLen = Math.max(maxLen, right - left);    // 更新答案
    }
    return maxLen;
}
```

**最小覆盖子串（字节高频）：**

```java
public String minWindow(String s, String t) {
    Map<Character, Integer> need = new HashMap<>(), window = new HashMap<>();
    for (char c : t.toCharArray()) need.merge(c, 1, Integer::sum);
    
    int left = 0, right = 0, valid = 0;
    int start = 0, minLen = Integer.MAX_VALUE;
    
    while (right < s.length()) {
        char c = s.charAt(right++);
        if (need.containsKey(c)) {
            window.merge(c, 1, Integer::sum);
            if (window.get(c).equals(need.get(c))) valid++;  // 该字符满足了
        }
        
        while (valid == need.size()) {       // 窗口已覆盖 t
            if (right - left < minLen) {     // 更新最小窗口
                minLen = right - left;
                start = left;
            }
            char d = s.charAt(left++);
            if (need.containsKey(d)) {
                if (window.get(d).equals(need.get(d))) valid--;
                window.merge(d, -1, Integer::sum);
            }
        }
    }
    return minLen == Integer.MAX_VALUE ? "" : s.substring(start, start + minLen);
}
```

```java
// 两数之和（有序数组）
public int[] twoSum(int[] numbers, int target) {
    int lo = 0, hi = numbers.length - 1;
    while (lo < hi) {
        int sum = numbers[lo] + numbers[hi];
        if      (sum == target) return new int[]{lo + 1, hi + 1};
        else if (sum < target)  lo++;
        else                    hi--;
    }
    return new int[]{};
}

public boolean isPalindrome(String s) {
    int lo = 0, hi = s.length() - 1;
    while (lo < hi) {
        while (lo < hi && !Character.isLetterOrDigit(s.charAt(lo))) lo++;
        while (lo < hi && !Character.isLetterOrDigit(s.charAt(hi))) hi--;
        if (Character.toLowerCase(s.charAt(lo)) != Character.toLowerCase(s.charAt(hi)))
            return false;
        lo++; hi--;
    }
    return true;
}
```

```java
public int removeDuplicates(int[] nums) {
    int slow = 0;
    for (int fast = 1; fast < nums.length; fast++) {
        if (nums[fast] != nums[slow]) {
            nums[++slow] = nums[fast];
        }
    }

}

// 三数之和（排序 + 固定一个 + 对撞指针）
public List<List<Integer>> threeSum(int[] nums) {
    Arrays.sort(nums);
    List<List<Integer>> res = new ArrayList<>();
    for (int i = 0; i < nums.length - 2; i++) {
        if (i > 0 && nums[i] == nums[i-1]) continue;  // 跳过重复
        int lo = i + 1, hi = nums.length - 1;
        while (lo < hi) {
            int sum = nums[i] + nums[lo] + nums[hi];
            if      (sum == 0) {
                res.add(Arrays.asList(nums[i], nums[lo], nums[hi]));
                while (lo < hi && nums[lo] == nums[lo+1]) lo++;
                while (lo < hi && nums[hi] == nums[hi-1]) hi--;
                lo++; hi--;
            }
            else if (sum < 0) lo++;
            else              hi--;
        }
    }
    return res;
}
```

> 滑动窗口用两个指针维护一个动态窗口，通过**扩大右边界**纳入新元素、**收缩左边界**排除不满足条件的元素，将 O(n²) 暴力优化到 O(n)。关键在于想清楚"什么条件下收缩左边界"以及"什么时候更新答案"（求最长→每轮扩大后更新；求最短→满足条件后在收缩前更新）。双指针适合有序数组的区间问题（Two Sum、三数之和）和原地操作（删除重复元素、移动零），利用已排序的单调性减少枚举。

```java
// 图的 BFS（求最短路径）
int bfs(List<List<Integer>> graph, int start, int end) {
    Queue<Integer> queue = new ArrayDeque<>();
    boolean[] visited = new boolean[graph.size()];
    queue.offer(start);
    visited[start] = true;
    int steps = 0;
    
    while (!queue.isEmpty()) {
        int size = queue.size();
        for (int i = 0; i < size; i++) {
            int node = queue.poll();
            if (node == end) return steps;
            for (int neighbor : graph.get(node)) {
                if (!visited[neighbor]) {
                    visited[neighbor] = true;
                    queue.offer(neighbor);
                }
            }
        }
        steps++;
    }
    return -1;
}
```

### 5. 拓扑排序（课程表，字节高频）

**Kahn 算法（BFS，入度表）：**

```java
// LeetCode 207: 课程表（判断是否有环）
public boolean canFinish(int numCourses, int[][] prerequisites) {
    List<List<Integer>> graph = new ArrayList<>();
    int[] inDegree = new int[numCourses];
    for (int i = 0; i < numCourses; i++) graph.add(new ArrayList<>());
    
    for (int[] pre : prerequisites) {
        graph.get(pre[1]).add(pre[0]);  // pre[1] → pre[0]
        inDegree[pre[0]]++;
    }
    
    Queue<Integer> queue = new ArrayDeque<>();
    for (int i = 0; i < numCourses; i++) {
        if (inDegree[i] == 0) queue.offer(i);  // 入度为0的先入队
    }
    
    int completed = 0;
    while (!queue.isEmpty()) {
        int course = queue.poll();
        completed++;
        for (int next : graph.get(course)) {
            if (--inDegree[next] == 0) queue.offer(next);  // 入度减为0则入队
        }
    }
    return completed == numCourses;  // 全部完成则无环
}
```

### 6. 并查集（Union-Find，连通性判断）

```java
// 路径压缩 + 按秩合并，均摊 O(α(n)) ≈ O(1)
class UnionFind {
    int[] parent, rank;
    int components;  // 连通分量数
    
    UnionFind(int n) {
        parent = new int[n]; rank = new int[n];
        components = n;
        for (int i = 0; i < n; i++) parent[i] = i;
    }
    
    int find(int x) {
        if (parent[x] != x) parent[x] = find(parent[x]);  // 路径压缩
        return parent[x];
    }
    
    boolean union(int x, int y) {
        int px = find(x), py = find(y);
        if (px == py) return false;  // 已经连通
        if (rank[px] < rank[py]) { int t = px; px = py; py = t; }
        parent[py] = px;
        if (rank[px] == rank[py]) rank[px]++;
        components--;
        return true;
    }
    
    boolean connected(int x, int y) { return find(x) == find(y); }
}
```

### 7. 面试标准答法

> BFS 用队列，按层扩展，适合求**最短路径**（无权图）和**层序遍历**；DFS 用栈（递归）或显式栈，沿路径深入，适合**连通性判断**、**拓扑排序**（DFS 后序反向）、**路径穷举**。拓扑排序用 Kahn 算法（BFS+入度表）更直观，能直接检测环（若最终处理节点数 < 总节点数则有环）。并查集以近乎 O(1) 的复杂度处理**动态连通性**问题（合并两个集合、查询两点是否连通），路径压缩+按秩合并后均摊复杂度达到 O(α(n))，α 是反阿克曼函数，实际可视为常数。

### 8. 常见追问

**Q: BFS 和 Dijkstra 最短路径的区别？**
> BFS 适用于**无权图**（每条边权重相同视为 1），通过层层扩展保证第一次到达某节点时就是最短路径，时间复杂度 O(V+E)。Dijkstra 适用于**有权图（非负权）**，用优先队列（最小堆）每次取距离最小的节点扩展，时间复杂度 O((V+E) log V)。Bellman-Ford 可处理**负权边**（O(VE)）但不能处理负权环。Floyd-Warshall 求**多源最短路**（O(V³)）。面试中图的最短路题通常用 BFS（无权）或 Dijkstra（有权）。
